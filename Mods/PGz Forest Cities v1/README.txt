PGz modlets are created and written by Phys1csGamez.

"PGz Forest Cities v1" is a modlet that brings back town and city generation in the forest and burnt forest biomes to 7 Days To Die v1.

Modlet installation:

1. Navigate to your C:\Users\USERNAME\AppData\Roaming\7DaysToDie
2. Create a folder called "Mods".
3. Extract or copy the contents of the zip file into the Mods folder. In the "Mods" folder, you will have a different folder for each mod you install. In that folder you will see the ModInfo.xml file.

Detailed video on how to install modlets https://www.youtube.com/watch?v=zhzw_EDqrYA
